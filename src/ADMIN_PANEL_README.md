# Админ-панель ZHEVAKA

## Доступ к админ-панели

Для демонстрации доступ к админ-панели осуществляется через кнопку "Админ-панель" в футере сайта.

## Основные разделы

### 1. Главная (Dashboard)
- **Общая статистика**: рецепты, пользователи, комментарии, просмотры
- **Задачи модерации**: количество элементов на модерации
- **Последняя активность**: лента событий на платформе
- **Быстрые действия**: переходы к основным разделам

### 2. Рецепты
- **Модерация рецептов**: просмотр и одобрение/отклонение рецептов от пользователей
- **Фильтрация**: по категориям и статусам (на модерации, одобренные, отклонённые)
- **Поиск**: по названию, автору и описанию
- **Просмотр деталей**: полная информация о рецепте перед модерацией
- **Удаление**: возможность удалить рецепт

### 3. Комментарии
- **Модерация комментариев**: просмотр всех комментариев с возможностью одобрения/отклонения
- **Фильтрация**: по статусам
- **Система жалоб**: отображение количества жалоб на комментарии
- **Поиск**: по автору, рецепту и содержанию

### 4. Пользователи
- **Управление пользователями**: просмотр всех пользователей платформы
- **Статистика активности**: рецепты, комментарии, избранное каждого пользователя
- **Блокировка/разблокировка**: управление доступом пользователей
- **Изменение ролей**: назначение модераторов
- **Фильтрация**: по ролям и статусам
- **Подробная информация**: детальный просмотр профиля пользователя

### 5. Реклама
- **Управление рекламными кампаниями**: создание, редактирование, удаление
- **Типы рекламы**: баннеры, боковые блоки, встроенная реклама, всплывающие окна
- **Статистика**: клики, показы, CTR, доход
- **Управление статусом**: активация/приостановка кампаний
- **Фильтрация**: по статусам

### 6. Аналитика
- **Посетители и просмотры**: графики динамики по дням
- **Популярность категорий**: диаграммы распределения рецептов
- **Устройства**: анализ по типам устройств (десктоп, мобильные, планшеты)
- **Источники трафика**: откуда приходят пользователи
- **Топ рецептов**: самые популярные рецепты с метриками
- **Топ пользователей**: самые активные авторы
- **Временные периоды**: выбор за сегодня/неделю/месяц/год

### 7. Настройки
#### Общие настройки
- Название и описание сайта
- Контактная информация (email, телефоны, адрес)

#### Управление функциями
- Пользовательские рецепты (вкл/выкл)
- Модерация (вкл/выкл)
- Комментарии (вкл/выкл)
- Оценки рецептов (вкл/выкл)
- Аналитика (вкл/выкл)
- Реклама (вкл/выкл)
- Режим обслуживания (вкл/выкл)

#### Настройки почты
- SMTP конфигурация
- Email уведомления
- Рассылки новостей

#### Настройки модерации
- Автоматическое одобрение (рецепты/комментарии)
- Ограничения длины рецептов
- Спам-фильтр
- Запрещённые слова

#### SEO настройки
- Meta теги (title, description, keywords)
- Google Analytics ID
- Яндекс.Метрика ID

## Функциональность

### Модерация
- Просмотр содержимого перед публикацией
- Одобрение/отклонение одним кликом
- Batch операции (планируется)

### Безопасность
- Разграничение прав доступа (админ/модератор/пользователь)
- Система блокировки пользователей
- Контроль за спамом

### Аналитика
- Визуализация данных через графики (Recharts)
- Экспорт данных (планируется)
- Сравнение периодов (планируется)

### Реклама
- Управление размещением
- Отслеживание эффективности
- Бюджетирование кампаний

## Технологии

- **React** - основной фреймворк
- **shadcn/ui** - компоненты UI
- **Recharts** - графики и диаграммы
- **Lucide React** - иконки
- **Sonner** - toast уведомления
- **Tailwind CSS** - стилизация
